# MBTI职业性格测评系统

## 系统介绍

MBTI职业性格测评系统是一个基于MBTI理论的在线测评工具，旨在帮助用户了解自己的性格类型，并获得相应的职业建议和个人发展指导。

## 功能特点

- **用户管理**：注册、登录、个人中心
- **测评功能**：完成MBTI测评问卷
- **结果分析**：查看测评结果和详细分析
- **数据导出**：导出测评记录为CSV或JSON格式
- **密码安全**：密码加密存储
- **数据验证**：表单输入验证
- **错误处理**：完善的错误提示
- **用户体验**：流畅的交互效果
- **服务器传输**：测评数据云端存储
- **图表展示**：测评结果数据可视化
- **AI分析**：基于DeepSeek API的智能分析报告

## 技术栈

- **前端**：Vue 3 + Vue Router
- **构建工具**：Vite
- **存储**：本地存储（localStorage）
- **数据加密**：Base64加密
- **HTTP客户端**：Axios
- **图表库**：Chart.js
- **AI服务**：DeepSeek API

## 安装说明

### 1. 克隆项目

```bash
git clone <repository-url>
cd zhMBTI
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

### 4. 构建生产版本

```bash
npm run build
```

## 使用方法

### 1. 注册账号

- 访问系统首页
- 点击导航栏中的"注册"按钮
- 填写注册信息并提交

### 2. 登录系统

- 访问系统首页
- 点击导航栏中的"登录"按钮
- 输入用户名和密码并登录

### 3. 开始测评

- 登录后点击"开始测评"按钮
- 回答测评问题
- 提交测评并查看结果

### 4. 查看个人中心

- 点击导航栏中的"个人中心"按钮
- 查看测评记录
- 导出测评数据

## 数据安全

- **密码加密**：密码使用Base64加密存储
- **本地存储**：所有数据存储在本地浏览器的localStorage中
- **数据隔离**：不同用户的数据相互隔离

- 系统仅使用用户数据进行测评和结果分析
