# MBTI职业性格测评系统使用手册

## 1. 系统介绍

MBTI职业性格测评系统是一个基于MBTI理论的在线测评工具，旨在帮助用户了解自己的性格类型，并获得相应的职业建议和个人发展指导。

### 1.1 系统功能

- **用户管理**：注册、登录、个人中心
- **测评功能**：完成MBTI测评问卷
- **结果分析**：查看测评结果和详细分析
- **数据导出**：导出测评记录为CSV或JSON格式
- **服务器传输**：测评数据云端存储
- **图表展示**：测评结果数据可视化
- **AI分析**：基于DeepSeek API的智能分析报告

### 1.2 技术架构

- **前端**：Vue 3 + Vue Router
- **存储**：本地存储（localStorage）
- **数据加密**：密码加密存储
- **HTTP客户端**：Axios
- **图表库**：Chart.js
- **AI服务**：DeepSeek API

## 2. 快速开始

### 2.1 注册账号

1. 访问系统首页
2. 点击导航栏中的"注册"按钮
3. 填写注册信息：
   - 用户名：至少3个字符
   - 密码：至少6个字符
   - 确认密码：与密码一致
   - 姓名：真实姓名
4. 点击"注册"按钮完成注册

### 2.2 登录系统

1. 访问系统首页
2. 点击导航栏中的"登录"按钮
3. 输入用户名和密码
4. 点击"登录"按钮完成登录

### 2.3 开始测评

1. 登录系统后，点击导航栏中的"开始测评"按钮
2. 系统将显示一系列测评问题，每个问题有5个选项：
   - 非常同意
   - 同意
   - 中立
   - 不同意
   - 非常不同意
3. 点击选项进行选择，系统会自动跳转到下一题
4. 完成所有问题后，点击"提交"按钮
5. 系统将计算测评结果并跳转到结果页面

## 3. 功能详解

### 3.1 个人中心

- **用户信息**：显示用户的基本信息，包括用户名、姓名和注册时间
- **测评记录**：显示用户的历史测评记录
- **导出功能**：点击"导出测评记录"按钮，可以将测评记录导出为CSV格式
- **退出登录**：点击"退出登录"按钮，退出当前账号

### 3.2 测评结果

- **性格类型**：显示用户的MBTI性格类型代码和描述
- **维度分析**：显示用户在四个维度上的得分情况，包括图表可视化展示
- **职业建议**：根据用户的性格类型，提供适合的职业建议
- **AI分析报告**：点击"AI分析"按钮，基于DeepSeek API获取智能分析报告，包括用户的优势、劣势、职业发展路径和个人成长建议

### 3.3 数据导出

- **CSV格式**：导出所有测评记录为CSV文件，包含详细的测评数据
- **JSON格式**：导出单个测评记录为JSON文件，包含完整的测评信息

## 4. 常见问题

### 4.1 如何修改密码？

目前系统暂不支持修改密码功能，如有需要请联系管理员。

### 4.2 测评结果准确吗？

MBTI测评结果仅供参考，不能作为唯一的职业选择依据。测评结果会受到用户当前状态和回答准确性的影响。

### 4.3 如何删除测评记录？

目前系统暂不支持删除测评记录功能，所有测评记录会保存在本地存储中。

### 4.4 系统支持哪些浏览器？

系统支持所有现代浏览器，包括Chrome、Firefox、Safari和Edge等。

## 7. 隐私政策

### 7.1 数据存储

- 所有用户数据和测评记录存储在本地浏览器的localStorage中
- 数据不会上传到服务器，确保用户数据的安全性

### 7.2 密码安全

- 密码使用加密算法存储，不会以明文形式保存
- 登录状态仅保存在本地，不会发送到服务器

### 7.3 数据使用

- 系统仅使用用户数据进行测评和结果分析
- 不会将用户数据用于其他目的
